# Краткое описание

Проект создавался в рамках отбора на стажировку в Avito осень 2025. [Здесь](https://github.com/HuggySoul/advertisement-moderation.git) более подробная документация по проекту.

# Доступные скрипты

- Запуск в режиме разработки:

  ```sh
  npm run dev
  ```

- Сборка проекта:

  ```sh
  npm run build
  ```

- Запуск в режиме предварительного просмотра производственной сборки:

  ```sh
  npm run preview
  ```

- Запуск ESlint:
  ```sh
  npm run lint
  ```
